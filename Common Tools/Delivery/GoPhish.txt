# ================= GoPhish =================
# Download and run GoPhish
wget https://github.com/gophish/gophish/releases/download/v0.x.x/gophish-v0.x.x-linux-64bit.zip
unzip gophish-v0.x.x-linux-64bit.zip
cd gophish
./gophish
